## 📖 About

**HashLayzer** is a lightweight, professional desktop application that calculates cryptographic hashes (MD5, SHA-1, SHA-256, SHA-512) of any file and checks them against VirusTotal's database of 70+ antivirus engines to identify potential malware.

Perfect for security researchers, malware analysts, and anyone who needs to verify file integrity or check for known threats.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔢 **Multiple Hash Types** | MD5, SHA-1, SHA-256, SHA-512 |
| 🛡️ **VirusTotal Integration** | Check files against 70+ antivirus engines |
| 📁 **File Metadata** | View file size, type, creation/modification dates |
| 🧠 **Local Malware Database** | Quick detection of known malware |
| 📋 **Copy to Clipboard** | One-click copy of all hashes |
| 🚀 **Large File Support** | Handles GB-sized files efficiently |
| 🎨 **Clean Light Theme** | Professional, easy-to-read interface |
| 🔒 **Secure API Handling** | API keys stored locally |
| 📊 **Multi-tab Results** | Organized display of hash values, metadata, and threat intelligence |
